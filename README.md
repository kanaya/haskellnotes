# Haskellのノート

「Haskellのノート」はプログラミング言語Haskellの入門書である．それと同時に，本書はプログラミング言語を用いた代数構造の入門書でもある．プログラミングと代数構造の間には密接な関係があるが，特に関数型プログラミングを実践する時にはその関係を意識する必要が出てくる．本書はその両者を同時に解説することを試みる．

## 本書の構成

本書は3部構成になる．そのうち第1部を先に公開する．

1. 第1部では，基本的なプログラミングを学びながら，抽象数学への梯子を登る．
2. 【執筆中】第2部では，実用的なHaskellプログラミングについて，抽象数学からの解説を踏まえつつ実践する．
3. 【執筆中】第3部では，プログラミングにおける代数構造を深く解説する．

## バージョン番号とブランチの説明

### バージョン番号

バージョン番号は `x.y.z` とし `x` は大幅な書き換え，追記があった場合に更新する．小幅な書き換え，追記があった場合は `y` を更新する．エラーの修正があった場合は `z` を更新する．

### ブランチ

`main` ブランチに最新のリリースを置く．`release-x.y` ブランチにバージョン `x.y.*` のリリース準備版を置く．
